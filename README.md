# PRODIGY_CS_01
## Description
This project is a Python-based Caesar Cipher tool developed as part of the Prodigy InfoTech Cyber Security Internship. It allows users to encrypt and decrypt text by shifting characters based on a given key.

## Features
- Encrypt text using Caesar Cipher
- Decrypt encrypted text
- User input for message and shift value

## Technologies Used
- Python

## Note
This project is for educational purposes to understand basic encryption techniques.
